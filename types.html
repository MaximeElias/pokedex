Type Feu

Les Pokémon de type Feu maîtrisent des attaques basées sur les flammes et sont redoutables contre les types Plante, Insecte, et Acier.

Type Eau

Les Pokémon de type Eau sont polyvalents, souvent utilisés pour contrer les types Feu, Sol, et Roche grâce à leurs attaques aquatiques.

Type Plante

Les Pokémon de type Plante, en harmonie avec la nature, possèdent souvent des capacités de soin et sont efficaces contre les types Eau, Sol, et Roche.

Type Électrique

Les Pokémon de type Électrique sont rapides et manient la foudre avec une grande précision. Ils sont redoutables contre les types Eau et Vol.

Type Roche

Les Pokémon de type Roche sont connus pour leur grande résistance physique. Ils sont efficaces contre les types Feu, Vol, Insecte et Glace.

Type Psy

Les Pokémon de type Psy utilisent leurs pouvoirs mentaux pour attaquer. Ils sont forts contre les types Combat et Poison.

Type Insecte

Les Pokémon de type Insecte sont généralement petits mais rapides et efficaces contre les types Psy, Plante et Ténèbres.

Type Vol

Les Pokémon de type Vol excellent dans les combats aériens et ont l'avantage contre les types Combat, Insecte et Plante.

Type Sol

Les Pokémon de type Sol sont résistants aux attaques électriques et sont particulièrement efficaces contre les types Feu, Roche et Acier.

Type Glace

Les Pokémon de type Glace sont puissants contre les types Vol, Sol, et Dragon grâce à leurs attaques frigorifiques.

Type Combat

Les Pokémon de type Combat sont spécialisés dans les attaques physiques et sont forts contre les types Normal, Roche et Ténèbres.

Type Poison

Les Pokémon de type Poison sont souvent toxiques et sont efficaces contre les types Fée et Plante.

Type Spectre

Les Pokémon de type Spectre sont mystérieux et intangibles. Ils sont forts contre les types Psy et Spectre.

Type Dragon

Les Pokémon de type Dragon sont extrêmement puissants, connus pour leurs attaques destructrices et efficaces contre eux-mêmes.

Type Ténèbres

Les Pokémon de type Ténèbres sont rusés et dangereux, excellents contre les types Psy et Spectre.

Type Acier

Les Pokémon de type Acier sont résistants à de nombreuses attaques et particulièrement forts contre les types Fée, Roche et Glace.

Type Fée

Les Pokémon de type Fée sont mystiques et efficaces contre les types Dragon, Ténèbres et Combat.

Type Normal

Les Pokémon de type Normal sont équilibrés mais n'ont pas d'avantages particuliers. Cependant, ils sont immunisés contre les attaques de type Spectre.
