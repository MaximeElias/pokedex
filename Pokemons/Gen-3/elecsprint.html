Elecsprint


Caractéristiques

  • Nom : Elecsprint
  • Taille : 1.5 m
  • Poids : 40.2 kg
  • Catégorie : Pokémon Décharge
  • Type : Électrik

Description

Elecsprint est un Pokémon de type Électrik. Il court à une vitesse fulgurante, accumulant de l'électricité dans son corps à chaque foulée. Il est capable de générer des éclairs puissants lorsqu'il est excité.

Évolutions

Elecsprint évolue à partir de Dynavolt.
