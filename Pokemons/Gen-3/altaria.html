Altaria


Caractéristiques

  • Nom : Altaria
  • Taille : 1.1 m
  • Poids : 20 kg
  • Catégorie : Pokémon Céleste
  • Type : Dragon / Vol

Description

Altaria est un Pokémon de type Dragon et Vol. Avec son apparence de nuage, il est connu pour sa capacité à chanter avec une belle voix. Altaria peut également créer des tempêtes avec ses ailes.

Évolutions

Altaria évolue à partir de Tylton.
