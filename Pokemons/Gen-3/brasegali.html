Brasegali


Caractéristiques

  • Nom : Brasegali
  • Taille : 1.6 m
  • Poids : 55 kg
  • Catégorie : Pokémon Flamme
  • Type : Feu / Combat

Description

Brasegali est un Pokémon de type Feu et Combat. Il est connu pour sa rapidité et sa force. Brasegali utilise ses pattes puissantes pour donner des coups de pied dévastateurs.

Évolutions

Brasegali évolue à partir de Galifeu, qui évolue lui-même à partir de Pussifeu.
