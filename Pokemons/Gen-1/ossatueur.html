Ossatueur


Caractéristiques

  • Nom : Ossatueur
  • Taille : 1.0 m
  • Poids : 75.0 kg
  • Catégorie : Pokémon Squelette
  • Type : Sol/Spectre

Description

Ossatueur est un Pokémon mystérieux qui erre dans les déserts et les ruines. Ce Pokémon est connu pour sa capacité à se camoufler dans le sable, devenant presque invisible pour ses proies. Ossatueur est aussi capable d'absorber les âmes de ceux qui osent s'approcher trop près, ce qui lui confère une aura de terreur.

Évolutions

Ossatueur évolue à partir de Osselait au niveau 35.
