Mystherbe


Caractéristiques

  • Nom : Mystherbe
  • Taille : 0.5 m
  • Poids : 6.5 kg
  • Catégorie : Pokémon Plante
  • Type : Plante, Poison

Description

Mystherbe est un Pokémon de type Plante et Poison. Il se cache souvent dans les herbes hautes, utilisant ses feuilles pour se camoufler. Mystherbe est connu pour son parfum agréable qui attire les insectes.

Évolutions

Mystherbe évolue en Ortide au niveau 21, puis en Raflésia lorsqu'elle est exposée à une Pierre Nuit.
