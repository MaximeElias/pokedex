Hypnomade


Caractéristiques

  • Nom : Hypnomade
  • Taille : 1.6 m
  • Poids : 75 kg
  • Catégorie : Pokémon Hypnose
  • Type : Psy

Description

Hypnomade est un Pokémon de type Psy. Il est connu pour sa capacité à manipuler le sommeil des autres. Hypnomade utilise sa grande pendule pour endormir ses adversaires et les plonger dans un profond sommeil. On dit qu'il peut lire les rêves des personnes endormies.

Évolutions

Hypnomade évolue à partir de Soporifik au niveau 25.
