Mélodelfe


Caractéristiques

  • Nom : Mélodelfe
  • Taille : 1.3 m
  • Poids : 40 kg
  • Catégorie : Pokémon Fée
  • Type : Fée

Description

Mélodelfe est un Pokémon de type Fée réputé pour sa voix enchanteresse. Elle utilise ses chants pour apaiser ses alliés et a le pouvoir d'élever l'humeur de ceux qui l'entendent.

Évolutions

Mélodelfe évolue à partir de Mélofée lorsqu'elle est exposée à une Pierre Lune. Avant cela, Mélofée évolue de Mélo.
