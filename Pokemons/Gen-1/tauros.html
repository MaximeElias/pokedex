Tauros


Caractéristiques

  • Nom : Tauros
  • Taille : 1.4 m
  • Poids : 80 kg
  • Catégorie : Pokémon Taurus
  • Type : Normal

Description

Tauros est un Pokémon de type Normal. Connu pour sa grande force et sa nature farouche, il charge avec une grande vitesse et peut être extrêmement difficile à contrôler. Tauros est souvent trouvé dans les prairies et les champs ouverts, où il peut courir librement.

Évolutions

Tauros n'évolue pas, mais il peut être capturé dans son état naturel.

Formes

Tauros a plusieurs formes régionales, notamment :

Tauros de Paldea - Type : Normal, Combat
