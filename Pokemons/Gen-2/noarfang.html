Noarfang


Caractéristiques

  • Nom : Noarfang
  • Taille : 1.6 m
  • Poids : 40.8 kg
  • Catégorie : Pokémon Hibou
  • Type : Normal, Vol

Description

Noarfang est un Pokémon de type Normal et Vol. Il est l'évolution de Hoothoot. Noarfang est connu pour ses capacités d'observation exceptionnelles, notamment sa vision nocturne incroyable, qui lui permet de chasser efficacement dans l'obscurité. Ce Pokémon est aussi très silencieux en vol, ce qui lui permet de surprendre ses proies sans être repéré.

Évolutions

Noarfang est l'évolution de Hoothoot au niveau 20.
