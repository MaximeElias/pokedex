Cotovol


Caractéristiques

  • Nom : Cotovol
  • Taille : 0.7 m
  • Poids : 3.5 kg
  • Catégorie : Pokémon Plume
  • Type : Plante, Vol

Description

Cotovol est un Pokémon de type Plante et Vol. Il est connu pour ses couleurs vives et sa capacité à se camoufler dans la nature. Cotovol aime se cacher dans les arbres et se nourrit de nectar.

Évolutions

Cotovol évolue à partir de Granivol.
