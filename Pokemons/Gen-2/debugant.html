Debugant


Caractéristiques

  • Nom : Debugant
  • Taille : 1.2 m
  • Poids : 50.0 kg
  • Catégorie : Pokémon Débugage
  • Type : Normal

Description

Debugant est un Pokémon de type Normal. Il est connu pour sa capacité à résoudre les problèmes et à déboguer les erreurs. Debugant est souvent utilisé par les chercheurs pour analyser des données.

Évolutions

Debugant évolue en Kicklee, Tygnon et Kapoera.
