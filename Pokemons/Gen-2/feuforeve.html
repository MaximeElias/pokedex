Feuforêve


Caractéristiques

  • Nom : Feuforêve
  • Taille : 0.7 m
  • Poids : 1.0 kg
  • Catégorie : Pokémon Fantôme
  • Type : Spectre

Description

Feuforêve est un Pokémon de type Spectre. Il est connu pour hanter les forêts et les maisons abandonnées, où il effraie les passants en faisant léviter des objets. Feuforêve adore faire des farces, mais il peut aussi se montrer bienveillant envers les personnes qu'il apprécie.

Évolutions

Feuforêve évolue en Magirêve.
