Snubbull


Caractéristiques

  • Nom : Snubbull
  • Taille : 0.6 m
  • Poids : 7.8 kg
  • Catégorie : Pokémon Bully
  • Type : Fée

Description

Snubbull est un Pokémon de type Fée. Il est connu pour son apparence mignonne mais peut être très protecteur envers son dresseur. Snubbull adore jouer et se lie rapidement d'amitié avec ceux qui le traitent bien.

Évolutions

Snubbull évolue en Granbull.
