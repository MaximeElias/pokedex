Joliflor


Caractéristiques

  • Nom : Joliflor
  • Taille : 0.8 m
  • Poids : 8.0 kg
  • Catégorie : Pokémon Fleur
  • Type : Plante, Poison

Description

Joliflor est un Pokémon de type Plante et Poison. Il est connu pour ses pétales colorés qui dégagent un parfum enivrant. Joliflor utilise ce parfum pour attirer les pollinisateurs et se protéger des prédateurs.

Évolutions

Joliflor évolue à partir de Ortide, qui elle-même évolue à partir de Mystherbe.
