Leuphorie


Caractéristiques

  • Nom : Leuphorie
  • Taille : 1.1 m
  • Poids : 46.0 kg
  • Catégorie : Pokémon Heureux
  • Type : Normal

Description

Leuphorie est un Pokémon de type Normal. Symbole de bienveillance et de joie, Leuphorie est connu pour sa capacité à soigner les blessures et à répandre le bonheur.

Évolutions

Leuphorie évolue à partir de Leveinard, qui lui-même évolue à partir de Ptiravi.
