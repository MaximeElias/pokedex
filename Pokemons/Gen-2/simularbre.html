Simularbre


Caractéristiques

  • Nom : Simularbre
  • Taille : 1.5 m
  • Poids : 35.0 kg
  • Catégorie : Pokémon Arbre
  • Type : Plante

Description

Simularbre est un Pokémon de type Plante. Il est connu pour son apparence d'arbre, ce qui lui permet de se camoufler parfaitement dans son environnement. Simularbre est souvent trouvé dans les forêts denses et utilise ses racines pour se nourrir des nutriments du sol.

Évolutions

Simularbre évolue à partir de Manzaï au niveau 32.
