Insolourdo


Caractéristiques

  • Nom : Insolourdo
  • Taille : 1.2 m
  • Poids : 30.0 kg
  • Catégorie : Pokémon Piège
  • Type : Insecte, Ténèbres

Description

Insolourdo est un Pokémon de type Insecte et Ténèbres. Il est connu pour sa capacité à se camoufler dans l'obscurité et à piéger ses proies avec ses techniques sournoises. Insolourdo aime rester dans les zones sombres, attendant le moment opportun pour attaquer.

Évolutions

Insolourdo évolue en Deusolourdo.
