Limonde


Caractéristiques

  • Nom : Limonde
  • Taille : 0.4 m
  • Poids : 10 kg
  • Catégorie : Pokémon Insecte
  • Type : Insecte, Eau

Description

Limonde est un Pokémon de type Insecte et Eau. Il est connu pour son habileté à se camoufler dans son environnement aquatique, utilisant ses ailes translucides pour se déplacer discrètement à la surface de l'eau.

Évolutions

Limonde n'a pas d'évolution.

Formes de Galar

Limonde a une forme régionale dans Galar :

Limonde de Galar - Type : Insecte, Acier
