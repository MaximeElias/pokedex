Verpom


Caractéristiques

  • Nom : Verpom
  • Taille : 0.2 m
  • Poids : 0.4 kg
  • Catégorie : Pokémon Pomme
  • Type : Plante/Dragon

Description

Verpom est un Pokémon de type Plante/Dragon. Il se cache dans une pomme pour éviter ses prédateurs et se déplace lentement, prenant soin de se dissimuler sous les feuilles. Ses évolutions sont liées à la variété de pomme qu'il consomme.

Évolution

Verpom évolue en Pomdrapi avec une pomme spécifique, puis peut évoluer en Dratatin ou Pomdramour, selon le type de pomme utilisé.
