Ixon


Caractéristiques

  • Nom : Ixon
  • Taille : 0.6 m
  • Poids : 38.5 kg
  • Catégorie : Pokémon Raton
  • Type : Ténèbres / Normal

Description

Ixon est l'évolution de Linéon dans la région de Galar. Ce Pokémon est connu pour sa rapidité et son agilité, utilisant ses compétences de ruse pour surprendre ses ennemis. Son type Ténèbres lui confère des capacités uniques et lui permet de se camoufler dans l'ombre.

Évolutions

Ixon évolue à partir de Linéon, qui est lui-même l'évolution de Zigzaton.
