Détails de Grillepattes

  • Nom : Grillepattes
  • Taille : 0.8 m
  • Poids : 12.0 kg
  • Catégorie : Pokémon Grillon
  • Type : Insecte / Sol

Description

Grillepattes est un Pokémon de type Insecte et Sol. Il est connu pour ses pattes robustes qui lui permettent de creuser et de se déplacer rapidement dans le sol. Grillepattes se trouve souvent dans les champs et les forêts, où il se nourrit de racines et d'insectes.

Évolution

Grillepattes évolue en Scolocendre au niveau 34.
